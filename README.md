# EditorDeGrafo

É um programa feito para as disciplinas de Teoria de Grafos, Extrutura de Dados II e Projeto Interdiciplinar do curso de 
Ciência da Computação na Unesc, realizado pelos academicos: Emerson, Dimitry, Eduardo, Victor.

Deve-se riar um editor em que a pessoa possa entrar com um grafos de maneira formal. Este grafo deve ser armazenado nas três 
estruturas que são: matriz de adjacência, matriz de incidência e lista de adjacência. O editor deve permitir que a pessoa 
salve o grafo e depois recupere os grafos que já foram salvos. Além disso o editor vai permitir que se possa ver as  três 
estruturas de armazenamento de forma didática, isto é, conforme viram em sala de aula.
